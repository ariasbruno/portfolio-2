# Bruno Arias — Portfolio (v2 "Legacy")

Este repositorio contiene una versión de mi portfolio personal que fue desarrollada recientemente, pero rápidamente reemplazada por una nueva visión.

## ⚠️ Contexto de este proyecto
Este portfolio tuvo una vida muy corta. Aunque fue terminado técnicamente, decidí reemplazarlo casi de inmediato porque no terminaba de convencerme la dirección visual o estructural que estaba tomando. Sin embargo, lo comparto de forma pública porque contiene piezas y animaciones de las que me siento orgulloso y que podrían ser útiles para otros o para mi futuro yo.

**Mi portfolio actual se encuentra en: [brunoarias.dev](https://brunoarias.dev)**

## 💎 Lo que "valió la pena"
A pesar de ser reemplazado, este proyecto incluye:
- **Diseños de Casos de Estudio**: Implementaciones visuales detalladas para proyectos.
- **Micro-interacciones**: Sistema de cursor personalizado y transiciones de página suaves.
- **Hero Backgrounds**: Fondos de patrones de puntos (dots) personalizados para proyectos como **RC Commander**, **El Impostor** e **ImpoStar**.

## 🛠️ Stack Tecnológico
- **Core**: React 19 + TypeScript.
- **Estilos**: Tailwind CSS v4.
- **Animaciones**: GSAP & CSS Transitions.
- **Iconografía**: Lucide React & SVGs personalizados.
